# ObjectId Generator CLI

This package is a generator for MongoDB ObjectId's for output in terminal

# Instalation

```sh
git clone 'this repo'
cd objectid-generator-cli
npm i
sudo npm install -g ./
```

# Usage

Just type `oid` in terminal and get the output
